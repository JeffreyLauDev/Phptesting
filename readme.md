### Step 1: Database Migration
Please follow the following command steps to migrate databse and import seed data :

`cd server`
 `php artisan migrate`
 `php artisan db:seed	`
` php artisan serve`    
Make sure server started here http://127.0.0.1:8000
### Step 2: Run the App
Please follow the following command steps to migrate databse and import seed data :

 `cd client `
 `npm install`
 `npm run serve`

### Step 3: Have Fun!  
Visit http://localhost:8080
